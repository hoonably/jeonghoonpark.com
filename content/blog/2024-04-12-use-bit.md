<article class="page sans" id="331451cf-7b79-806d-b827-d5c8048084f3"><header><p class="page-description" dir="auto"></p></header><div class="page-body"><div dir="auto" style="display:contents"><h3 class="" id="331451cf-7b79-8071-92d5-d1e561b1f99e">2진수 표기</h3></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-80ae-a74b-d58c86fbe022">숫자 앞에 "0b" 를 붙인다.</p></div><div dir="auto" style="display:contents">

```cpp
int a = 0b101;  // a = 5 (1*4 + 0*2 + 1*1)
cout << a << endl;  // 5 출력
```

</div><div dir="auto" style="display:contents"><hr id="331451cf-7b79-8069-9c05-e42c457075a2"/></div><div dir="auto" style="display:contents"><h3 class="" id="331451cf-7b79-8061-b764-c4777be0f210">2진수로 출력 (C++)</h3></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-80ce-b1ff-ea0ea9ffffe9">헤더파일 : #include &lt;bitset&gt;</p></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-8040-850c-fdbe7f5de9a4">bitset&lt;비트의 개수&gt;(출력할 변수)</p></div><div dir="auto" style="display:contents">

```cpp
char c = 2;  // char은 8비트
int i = 5;  // int는 32비트

cout << bitset<1>(c) << endl;  // 0 출력 (제대로 된 값을 알 수 없음)
cout << bitset<8>(c) << endl;  // 00000010 출력
cout << bitset<4>(i) << endl;  // 0101 출력
cout << bitset<32>(i) << endl;  // 00000000000000000000000000000101 출력

// string 이용하여 저장
string bit = bitset<8>(i).to_string();
cout << bit << endl;  // 00000101 출력
```

</div><div dir="auto" style="display:contents"><hr id="331451cf-7b79-80b8-9b41-f8a50d659e96"/></div><div dir="auto" style="display:contents"><h3 class="" id="331451cf-7b79-8092-af6d-d10a1cfefb15">8진수 표기</h3></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-8053-a7b2-df7a3306db22">숫자 앞에 "0" 를 붙인다.</p></div><div dir="auto" style="display:contents">

```cpp
int a = 0107;  // a = 71 (1*64 + 0*8 + 7*1)
cout << a << endl;  // 71 출력
```

</div><div dir="auto" style="display:contents"><hr id="331451cf-7b79-80d2-8567-cd156d2371bc"/></div><div dir="auto" style="display:contents"><h3 class="" id="331451cf-7b79-80a5-a52a-c1eebd2da73e">16진수 표기</h3></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-804f-8184-d48a6505c9da">숫자 앞에 "0x" 를 붙인다.</p></div><div dir="auto" style="display:contents">

```cpp
int a = 0x3a;  // a = 58 (3*16 + 10*1)
cout << a << endl;  // 58 출력
```

</div><div dir="auto" style="display:contents"><hr id="331451cf-7b79-8016-ad5e-ce90a7652ad3"/></div><div dir="auto" style="display:contents"><h3 class="" id="331451cf-7b79-80ed-83d1-f379865db5e5">효과</h3></div><div dir="auto" style="display:contents"><ul class="bulleted-list" id="331451cf-7b79-80df-98b8-e5e925c4a507"><li style="list-style-type:disc">C 언어에서 file을 open할때 다음과 같은 8진수 식을 사용하여 쉽게 mode를 표기할 수 있다.</li></ul></div><div dir="auto" style="display:contents">

```cpp
fd = open("hello.txt", O_CREAT | O_EXCL, 0644);  // "0숫자" 는 8진수를 표기하는 방식
```

</div><div dir="auto" style="display:contents"><ul class="bulleted-list" id="331451cf-7b79-80ff-99e6-f1eb7632108b"><li style="list-style-type:disc">비트마스킹을 이용한 다이나믹 프로그래밍에서는 자신의 코드가 잘 돌아가고 있는지 파악하기 힘든데,<div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-805c-913e-ceae3f63a460">2진수를 직접 입력해 테스트해볼 수 있다.</p></div><div dir="auto" style="display:contents"><p class="" id="331451cf-7b79-809a-9ed3-e2e15ad0116f"><a href="https://www.notion.so/blog/ps-1062/">EX) 백준 가르침 비트마스킹 풀이</a></p></div></li></ul></div><div dir="auto" style="display:contents">

```cpp
cout << dp[0b0000111];  // 0,1,2 번 비트가 켜져있는 dp 출력해보기
```
